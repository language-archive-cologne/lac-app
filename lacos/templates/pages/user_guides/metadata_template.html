The Language Archive Cologne provides an Excel-based metadata template for depositing materials. This excel sheet allows you to enter metadata in a way that allows us to transfer it into the archive.

Citation

Saleh, Sara, & Granderath-Miegel, Cosima. (2024). Metadata Template for Language Archive Cologne (1.0.1). Zenodo. https://doi.org/10.5281/zenodo.7568575

Key Sections to Complete

1. General Information

Researchers enter recurring details here, including descriptions of all contributors referenced by code throughout other sections.

2. Collection-Level Data

This will provide information for the top-level-node of your archive describing the collection as a whole, beginning with the CollectionDisplayTitle column.

3. Bundles

Each row represents grouped files with metadata, starting from the MetadataFilename column. Access restrictions are specified at the bundle level, offering options for:

  • Open access
  • Registration-required
  • Restricted access

Purpose

The template standardizes how researchers document linguistic and cultural materials for long-term preservation and discoverability within the archive system.
