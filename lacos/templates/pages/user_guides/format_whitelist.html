The Language Archive Cologne maintains an approved formats list for submissions, organized by data type in order of preference.

Metadata

  • BLAM CMDI

Audio

WAV with Linear Pulse Code Modulation (LPCM) audio (preferred sampling rate 48 kHz and bit depth 16 bit)

Video

  • MP4: h.264 codec (main profile, level 4.0, 1080p, 30fps), AAC audio (48 kHz, 128-384 kbps)
  • MOV: h.264 codec (main profile, level 4.0, 1080p, 30fps), LPCM audio (48 kHz, 16 bit)

Annotations

  • ELAN
  • Praat TextGrid
  • Exmaralda transcriptions
  • TEI (per ISO 24624:2016)
  • FLeX XML
  • Toolbox Files

Written Resources

  • Plain UTF-8 text files
  • PDF/A
  • XHTML

Still Images

Listed in order of preference:

  • TIFF
  • JPEG2000
  • PNG
  • JPEG

Additional Metadata

  • CMDI metadata
  • XML formats
  • CSV with optional W3C Metadata Vocabulary annotations
